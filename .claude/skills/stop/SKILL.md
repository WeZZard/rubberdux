# Stop Rubberdux

Stop the running rubberdux process and clean up artifacts generated by /launch (log file, background task outputs, subagent sessions, tool results).

Run the stop script:

```
${CLAUDE_SKILL_DIR}/scripts/stop.sh
```

After the script completes, report:
- Whether a process was stopped
- How many artifacts were cleaned
